# 🏭 FactoryIQ

### AI-Powered Industrial Knowledge Intelligence Platform

FactoryIQ is an enterprise-grade AI-powered platform that centralizes industrial knowledge, streamlines equipment maintenance, and enhances compliance management. It leverages Artificial Intelligence, Retrieval-Augmented Generation (RAG), Semantic Search, and Knowledge Intelligence to help organizations make faster and smarter operational decisions.

🌐 **Live Demo:** https://factoryiq-1-j3y2.onrender.com

---

# 🚀 Overview

Industries generate thousands of documents including:

- Equipment Manuals
- Standard Operating Procedures (SOPs)
- Maintenance Reports
- Inspection Reports
- Safety Guidelines
- Compliance Documents

Finding the right information during equipment failures or audits is time-consuming.

FactoryIQ solves this problem by converting scattered industrial documents into an intelligent AI-powered knowledge platform.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/your-username/factoryiq.git
```

---

## Frontend

```bash
cd frontend

npm install

npm run dev
```

---

## Backend

```bash
cd backend

npm install

npm run dev
```

---

# 🔑 Environment Variables

Create a `.env` file inside the backend directory.

```env
PORT=5000

MONGODB_URI=your_mongodb_connection

JWT_SECRET=your_secret_key

GEMINI_API_KEY=your_google_gemini_api_key
```
